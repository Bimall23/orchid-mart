# Orchid Local Mart UI

This repository contains a lightweight Angular-based prototype for the
"Orchid Local Mart" e-commerce platform described in `E.docx`. All data
is static and intended only to demonstrate the front-end design.

## Features

- Product listing with search and category filters
- Shopping cart with quantity tracking
- Basic checkout modal with multiple payment options
- Loyalty discount automatically applied to orders over $1000
- Optional subscription to remove shipping fees
- Orders page with status tracking and return requests
- Inventory counts update as orders are placed
- Responsive layout using AngularJS (Angular 2+ style via CDN)

Open `src/index.html` in a browser to try the demo.
